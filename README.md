# Big Data Analytics

This repository contains hands-on Big Data Analytics projects completed as part
of the PGDM (Finance & Analytics) program at IMT Hyderabad.

## Areas Covered
- Unsupervised Learning (Clustering)
- Sentiment Analysis using Transfer Learning
- Large-scale text analytics
- Business interpretation of analytical outputs

## Tools & Techniques
- Machine Learning
- NLP & Transfer Learning
- Excel / Python-based analytics
- Model evaluation & interpretation

Each folder inside this repository represents an individual project with
datasets, analysis, and business insights.
